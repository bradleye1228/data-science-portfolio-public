# README for: 02-patient-health-analytics

## Overview
This R Shiny application was developed as part of DATA423 at the University of Canterbury. It focuses on exploratory and predictive analysis of patient health data using interactive visualisations and machine learning workflows.

## Objectives
The purpose of this dashboard is to explore relationships between patient health variables and support interpretation of patterns relevant to health outcomes. The application enables dynamic exploration of data distributions and model-based insights.

## Key Features
- Interactive patient health data exploration
- Visualisation of health indicators and distributions
- Missing data and data quality assessment tools
- Correlation and multivariate analysis
- Integration of predictive modelling outputs
- Interactive filtering and parameter controls

## Analytical Focus
The dashboard enables investigation of:
- Health variable relationships and dependencies
- Data completeness and quality issues
- Predictive signal within health features
- Model comparison insights (where applicable)

## Tools & Technologies
- R
- Shiny
- Plotly
- caret / tidymodels
- GGally, visdat

## Outcome
This dashboard demonstrates how interactive analytics and modelling can support exploratory insights into healthcare-related datasets and highlight variables of predictive importance.
