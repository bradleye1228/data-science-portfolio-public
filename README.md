# README for: 01-manufacturing-dashboard

## Overview
This interactive R Shiny application was developed as part of DATA423 at the University of Canterbury. It explores a structured dataset using exploratory data analysis techniques and interactive visualisations to support decision-making in a manufacturing context.

## Objectives
The aim of this application is to identify key relationships, distributions, and anomalies within the dataset through interactive visual exploration. The dashboard enables users to investigate variables dynamically and uncover patterns that may not be visible through static analysis.

## Key Features
- Interactive exploratory data analysis (EDA) dashboard
- Boxplots with adjustable outlier thresholds and scaling controls
- Correlation analysis of numeric variables
- Mosaic plots for categorical relationships
- Missing data visualisation
- Interactive data tables for inspection
- Rising-value and distribution-based visual diagnostics
- User-controlled filtering and chart parameterisation

## Analytical Focus
The dashboard supports identification of:
- Variable distributions and spread
- Potential outliers and data quality issues
- Correlation structure between features
- Data completeness and missingness patterns

## Tools & Technologies
- R
- Shiny
- Plotly
- vcd, GGally, corrgram, visdat, DT

## Outcome
This application demonstrates how interactive visual analytics can enhance exploratory understanding of structured datasets and guide further statistical investigation.
